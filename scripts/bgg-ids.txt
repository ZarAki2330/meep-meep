# Jeux à intégrer au catalogue « tout prêt ».
# Une entrée par ligne. Tout ce qui suit un « # » est ignoré.
#
# Chaque ligne peut être :
#   - un identifiant BGG numérique (dans l'URL : .../boardgame/266192/... -> 266192), ou
#   - un NOM de jeu, résolu automatiquement en id via la recherche BGG.
#
# Lancer :  node scripts/generer-catalogue.mjs
#     ou :  node scripts/generer-catalogue.mjs --hot   (ajoute les jeux du moment, liste « Hot » BGG)
#
# Les jeux déjà présents dans catalogue.json sont ignorés (rien n'est écrasé).
# Le script affiche, pour chaque nom, l'id et le jeu retenus : vérifie-les.
#
# ─────────────────────────────────────────────────────────────────────────────
# Sélection de ~95 titres POPULAIRES EN FRANCE, pour compléter la bibliothèque
# déjà livrée dans l'app. Sources : meilleures ventes Philibert 2025, lauréats
# As d'Or (Cannes), classiques grand public et valeurs sûres du hobby.
#
# NB : ce n'est PAS un « top 1000 » (cette donnée n'existe pas de façon fiable) —
# c'est une base large à ajuster selon tes goûts. Retire les lignes non voulues.
# Les jeux déjà inclus dans l'app (Catan, Carcassonne, 7 Wonders, Pandémie,
# Azul, Splendor, Dixit, Codenames, Just One, Time's Up, The Crew, Skull King,
# Villainous, 6 qui prend, Mille Bornes, Scrabble, Yams, Belote) sont omis.
#
# Astuce séries (Unlock!, Exit, Clank!, Marvel Champions…) : la recherche peut
# renvoyer le jeu générique. Vérifie l'id affiché, ou mets l'id d'un opus précis.
# ─────────────────────────────────────────────────────────────────────────────
